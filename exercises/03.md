Surasti h1 antraste
h1 antrastei prideti CSS class "simba"

Surasti visas darzoves
Kiekvienai darzovei prideti CSS class "darzove"

```html
<h1>Sriubos receptas</h1>
<ul>
    <li>Morkos</li>
    <li>Bulves</li>
    <li>Svogunas</li>
    <li>Vanduo</li>
    <li>Druska</li>
</ul>
```

```css
.simba {
    color: red;
    text-transform: uppercase;
}

.darzove {
    font-style: italic;
    font-size: 20px;
}
```
